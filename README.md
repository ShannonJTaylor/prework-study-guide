# <Prework Study Guide Webpage>

## Description 

This webpage is designed to be a repository of notes taken during the prework of coding bootcamp. This webpage was built as a study guide to make it easy to access and share information. This improves efficiencies and allows for the opportunity to demonstrate the skills learned throughtout the prework.

## Usage

This guide is organized in sections, each section concentrating on a certain topic like HTML, CSS, Git, and JavaScript. Opening the DevTools console on the webpage allows us the opportunity to see the JavaScript running. This script will list the topics we learned from prework and offer you suggestions of which one to study first.
